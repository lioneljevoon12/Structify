Apa time complexity dari operasi insertHead() pada Singly Linked List?
Petunjuk
Perhatikan: apakah operasi ini perlu traverse seluruh list?
A
O(n) — harus traverse sampai akhir list
B
O(1) — hanya update pointer head saja
C
O(log n) — menggunakan binary search
D
O(n²) — nested loop untuk posisi
Jawaban Benar!

InsertHead hanya memerlukan dua operasi pointer: set new_node.next = head, lalu head = new_node. Tidak ada traversal — kompleksitasnya O(1) konstan.
